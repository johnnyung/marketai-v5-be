# PHASE 3 — INTELLIGENCE OVERVIEW

Phase 3 introduces **interpretive intelligence**, not ingestion.

## Purpose
Transform existing Phase 1–2 data into explainable investment opportunities.

## Constraints
- No new ingestion
- No pipeline refactors
- No frontend changes
- Read-only consumption of existing data

## Outputs
- Opportunity objects
- Confidence scoring
- Risk flags
- Explainability payloads
